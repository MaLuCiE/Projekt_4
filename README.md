# Projekt_4
Správce úkolů (Task Manager)

Jednoduchá konzolová aplikace napsaná v **Pythonu**, která slouží ke **správě úkolů**.  
Umožňuje přidávat, zobrazovat a odstraňovat úkoly prostřednictvím jednoduchého menu.
Tento projekt vznikl v rámci **rekvalifikačního kurzu Testing akademie** jako cvičení na tvorbu a testování menší aplikace.

# Funkce

- **Hlavní menu s volbou akcí**
- **Přidání úkolu**
- **Odstranění úkolu**
- **Zobrazení všech úkolů**
- **Ukončení programu**

# Použité technologie

- **Python 3**
